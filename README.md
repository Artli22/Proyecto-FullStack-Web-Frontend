# Bienvenido al repositorio Frontend del proyecto Full Stack
## Aqui se guardaron aquellos archivos que consistuyen a la UI del proyecto

### Funcionamiento 
- Visualizar todas las series dentro de la base de datos series2.
- Agregar series dentro de la base de datos y verlas en la ui.
- Modificar datos de series ya ingresadas o recientemente creadas.
- Usar los controles de paginacion ubicados en la parte baja de la pantalla.
- Orendar la lista de series por ID, Nombre, episodio actual o episodios totales.
  
### Tecnologias Implementadas
- JavaScript
- HTML
- CSS
